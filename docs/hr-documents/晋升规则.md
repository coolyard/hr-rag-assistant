# 晋升规则

## 晋升评估时间

- 员工每年有一次晋升评估机会
- 评估时间：每年12月

## 晋升考核标准

晋升考核包含三个维度：

1. 业绩评估（50%权重）：年度工作目标和关键业绩指标完成情况
2. 能力评估（30%权重）：专业能力、领导力、创新能力
3. 价值观评估（20%权重）：公司核心价值观践行情况

## 晋升等级

- P4 → P5：初级工程师/专员 → 中级工程师/专员
- P5 → P6：中级 → 高级
- P6 → P7：高级 → 资深
- M1 → M2：团队负责人 → 部门负责人

## 晋升后薪资调整

- 晋升后薪资调整幅度：15%-30%
- 调整后的薪资从次月开始生效

## 晋升申请流程

1. 员工自评：填写年度工作总结和能力提升计划
2. 直属上级评审：给出晋升建议和评价
3. 跨部门评审：相关协作部门给出评价意见
4. 晋升委员会决策：综合评估后给出最终决策

## 试用期规定

- 试用期员工不可参与晋升评估
- 转正满一年后即可申请下一年度的晋升评估

## 其他说明

- 晋升评估结果通常在次年1月公布
- 如对评估结果有异议，可在结果公布后5个工作日内提出申诉
